# Receipt Report Automation

Automates the extraction of daily receipt reports from Proleaz leasing software and pushes clean data to Google Sheets for real-time management review.

## Tech Stack
- Python, Selenium, Pandas, Google Sheets API

## What it does
- Logs into Proleaz and extracts daily receipt data using Selenium
- Cleans and transforms data (date formatting, deduplication) using Pandas
- Pushes structured results to Google Sheets via API

## Impact
Saves 15-20 minutes of manual work every day.
